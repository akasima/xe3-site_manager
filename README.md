# XE3 Site Manager
이 어플리케이션은 Xpressengine3(이하 XE3)의 플러그인입니다.

## 설명
이 플러그인은 XE3로 웹사이트를 운영하는데 필요한 관리자 기능을 제공합니다.

서버 상태를 확인하고, 캐시 삭제, 로그파일 삭제를 지원합니다.
문제를 해결하기 위한 간편한 기능을 제공합니다.

웹사이트를 운영하면서 발생하는 문제를 해결하고 서비스 상태를 모니터링 할 수 있는 기능을 제공합니다.
